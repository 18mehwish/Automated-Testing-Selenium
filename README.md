\# Automated Testing Framework (Selenium)



\## Overview

This project demonstrates a basic automated testing framework using Selenium.



\## Features

\- Automates browser actions

\- Performs search testing on Google

\- Can be extended for web app testing



\## Tech Stack

\- Python

\- Selenium



\## Future Improvements

\- Add PyTest integration

\- Generate test reports

\- Screenshot on failure



\## How to Run

1\. Install dependencies:

&#x20;  pip install selenium



2\. Run script:

&#x20;  python main.py
